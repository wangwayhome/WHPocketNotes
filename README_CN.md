# WHPocketNotes

[English](README.md) | [中文](README_CN.md)

一个轻量级、注重隐私的 iOS 笔记应用，让您的想法安全且可离线访问。

![iOS](https://img.shields.io/badge/iOS-13.0%2B-blue)
![Swift](https://img.shields.io/badge/Swift-5.0-orange)
![License](https://img.shields.io/badge/License-MIT-green)

## 📝 关于

WHPocketNotes 是一款简单高效的笔记应用，专为重视隐私和简洁性的 iOS 用户设计。无需注册账号，所有数据都存储在本地设备上，您可以快速记录和整理想法，无需担心数据隐私或网络连接问题。

## ✨ 功能特点

- **📱 快速创建笔记** - 通过简单直观的界面即时创建和保存笔记
- **🔍 智能搜索** - 使用关键词搜索功能快速找到您的笔记
- **📋 有序列表视图** - 在简洁易用的列表中查看所有笔记
- **🗑️ 轻松管理** - 通过简单的滑动或点击删除笔记
- **🔒 隐私优先** - 无需账号，无云同步，不收集数据
- **📴 离线优先** - 所有笔记存储在本地设备上，完全离线访问
- **⚡ 快速轻量** - 最小化资源占用，优化性能表现

## 🚀 快速开始

### 系统要求

- iOS 13.0 或更高版本
- Xcode 12.0 或更高版本（用于开发）
- CocoaPods（用于依赖管理）

### 安装步骤

1. 克隆仓库：
```bash
git clone https://github.com/wangwayhome/WHPocketNotes.git
```

2. 进入项目目录：
```bash
cd WHPocketNotes/PocketNotes
```

3. 安装依赖：
```bash
pod install
```

4. 打开工作空间：
```bash
open PocketNotes.xcworkspace
```

5. 在 Xcode 中构建并运行项目。

## 🔐 隐私与安全

您的隐私是我们的首要任务：

- ✅ **无需用户账号** - 无需注册即可立即开始使用
- ✅ **仅本地存储** - 所有数据都保留在您的设备上
- ✅ **无需网络** - 完全离线工作
- ✅ **不收集数据** - 我们不收集、存储或共享任何个人信息
- ✅ **无第三方服务** - 没有分析、广告或跟踪

更多详情，请参阅我们的[隐私政策](PrivacyPolicy.md)。

## 🛠️ 技术栈

- **语言**: Swift
- **平台**: iOS
- **架构**: MVC（模型-视图-控制器）
- **依赖管理器**: CocoaPods
- **数据存储**: 本地设备存储（Core Data / UserDefaults）

## 📱 使用方法

1. **创建笔记**：点击"+"按钮创建新笔记
2. **编辑笔记**：点击任何笔记以查看和编辑其内容
3. **搜索笔记**：使用搜索栏通过关键词查找特定笔记
4. **删除笔记**：在笔记上向左滑动或点击删除按钮
